# HƯỚNG DẪN SỬ DỤNG

## 1. Giới thiệu
Ứng dụng quản lý và đặt lịch khám bệnh trực tuyến viết bằng JavaFX.

## 2. Yêu cầu hệ thống
- Java JDK 21
- JavaFX SDK 24
- Hệ điều hành Windows

## 3. Cách chạy chương trình
1. Clone project từ GitHub
2. Mở bằng IntelliJ IDEA
3. Chạy file Main.java
